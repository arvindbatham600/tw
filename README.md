## Run in Your Local

### Clone the Repo

```bash
git clone https://github.com/arvindbatham600/tw.git
```

go to sm-project folder

```bash
cd sm-project
```



Install the dependencies
```bash
yarn
```


Run the Server

```bash
yarn dev
```
